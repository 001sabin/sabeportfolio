# Sabin Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Includes dark/light mode, sections for Hero, About, Skills, Projects, Resume, Contact, and a Blog placeholder.

## Quick start
- Open `index.html` in your browser.
- Edit text and links directly in `index.html`.
- Styles: `assets/css/styles.css`
- Scripts: `assets/js/main.js`

## Customize
- Replace avatar: put an image at `assets/img/avatar.svg` or `avatar.png` and update the `<img>` path.
- Projects: edit cards in the Projects section.
- Resume: replace `assets/resume/Sabin_Shrestha_Resume.pdf` with your PDF.
- Theme: adjust CSS variables at the top of `styles.css`.

## Deploy
Host on GitHub Pages, Netlify, or Vercel by uploading the folder.
